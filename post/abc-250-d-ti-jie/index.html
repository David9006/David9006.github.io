ABC250D 题解

2024-02-16 OI

（vjudge 题面：https://vjmirror.link/problem/AtCoder-abc250_d）

题意简述

给定整数 NNN（1≤N≤10181 \le N \le 10^{18}1≤N≤1018），求出所有满足以下性质的整数的个数：设该整数为 xxx，则

  • x≤Nx \le Nx≤N，
  • 存在素数 p,qp, qp,q 使得 x=p⋅q3x = p \cdot q^3x=p⋅q3 且 p<qp < qp<q。

解法

可以发现，若存在素数 p,qp, qp,q 使得 N=p⋅q3N = p \cdot q^3N=p⋅q3，则 q≤N3≤106q \le \sqrt[3]{N} \le 10^6q≤3N​≤106。那么我们可以预处理 N3\sqrt[3]{N}3N​ 以内的素数。此时，因为 10610^6106 以内的素数大约只有 106ln⁡106≈76382\dfrac{10^6}{\ln 10^6} \approx 76382ln106106​≈76382 个，所以我们可以枚举 p,qp, qp,q 来生成满足性质的整数，同时若生成出来的大于 NNN，则停止。

时间复杂度：O(能过)O(\text{能过})O(能过)

代码

该代码使用埃氏筛 + bitset 筛素数，获得了 15 ms 通过的好成绩。

#include <bitset>
#include <ctime>
#include <iostream>
#include <vector>

#define endl (char)('\n')

using namespace std;

using u64 = unsigned long long;
using i64 = long long;

constexpr int MAX_N = 1e6;

vector<int> primes;

void get_primes()
{
    bitset<MAX_N + 5> vis(3);

    for (int i = 2; i <= MAX_N; ++i) {
        if (vis[i])
            continue;

        primes.emplace_back(i);

        for (i64 j = 2; i * j <= MAX_N; ++j)
            vis[i * j] = true;
    }
}

int main()
{
    get_primes();

    u64 n;
    cin >> n;

    i64 cnt = 0;

    for (u64 i = 0; i < primes.size(); ++i) {
        for (u64 j = i + 1; j < primes.size(); ++j) {
            double p = primes[i], q = primes[j];

            if (p * q * q * q <= n)
                ++cnt;
            else
                break;
        }
    }

    cout << cnt << endl;
    return 0;
}
