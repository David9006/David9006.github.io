ABC330E 题解

2024-02-16 OI

（原题面：https://vjmirror.link/problem/AtCoder-abc330_e）

题意简述

给定 NNN（1≤N≤2×1051 \le N \le 2 \times 10^51≤N≤2×105）个整数 A1,A2,...,ANA_1, A_2, ..., A_NA1​,A2​,...,AN​ 和 QQQ（1≤Q≤2×1051 \le Q \le 2 \times 10^51≤Q≤2×105）次询问。询问以 i x（1≤i≤N,1≤x≤1091 \le i \le N, 1 \le x \le 10^91≤i≤N,1≤x≤109）的格式给出，我们要

  • 先将 AiA_iAi​ 修改为 xxx，
  • 后输出最小的不在这 NNN 个数当中的非负整数（简记为 mex⁡(A)\operatorname{mex}(A)mex(A)）。

解法

我们维护每个数出现的次数 cic_ici​ 和一个集合 SSS，存储 [0,N+1][0, N+1][0,N+1] 中所有不在 AAA 中的整数。则 mex⁡(A)\operatorname{mex}(A)mex(A) 为 SSS 中的最小值。在处理询问时，若原来的数 AiA_iAi​ 在修改后不在 AAA 中，就把它加入 SSS，同时若修改后的数 xxx 在修改前不在 AAA 中，就把 xxx 从 SSS 删除。

时间复杂度：若使用 std::set 实现，为 O(Qlog⁡N)O(Q \log N)O(QlogN)，可以通过。

实现

该代码同时使用 std::unordered_map 实现（所以常数较大），因为我不想离散化。

#include <iostream>
#include <set>
#include <unordered_map>
#include <vector>

using namespace std;
using i64 = long long;

int main()
{
    int n, q;
    vector<int> arr;
    unordered_map<int, int> cnts;

    cin >> n >> q;

    for (int i = 0; i < n; ++i) {
        int x;
        cin >> x;

        ++cnts[x];
        arr.emplace_back(x);
    }

    set<int> unfound;

    for (int i = 0; i <= n; ++i) {
        if (cnts[i] == 0)
            unfound.emplace(i);
    }

    while (q-- > 0) {
        int i, k;
        cin >> i >> k;

        --i;
        --cnts[arr[i]];

        if (cnts[arr[i]] == 0)
            unfound.emplace(arr[i]);

        if (cnts[k] == 0)
            unfound.erase(k);

        ++cnts[arr[i] = k];
        cout << *unfound.begin() << '\n';
    }
    
    return 0;
}
